Type detail is an ongoing project by Wenting Zhang, inspired by the great discontent's 100 day project and Stephen Coles's great book The Anatomy of Type. The rule and goal are to annotate a web typeface each day, pointing out the beautiful details of the type forms that often get overlooked. This project is going to last for 100 days between May 18 and Aug 26. Anatomy of a Typeface by typedia explains typography terminology used in this project. If you are interested in joining me on this type exploratory trip, you could grab my template and start making annotation of your favorite typefaces.
